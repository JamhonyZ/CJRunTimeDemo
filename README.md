# CJRunTimeDemo
UIButton分类，常用功能封装。
支持block回调，支持按钮间隔时长设置，支持图片文字frame设置，只是热响应区域设置

UIView分类。
支持block手势单击和长按。支持占位功能，开放常用无网络，无数据，加载失败 三种常用类型。开放自定义图片和提示文字。具体可参考demo。
